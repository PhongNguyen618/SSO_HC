Cài Đặt API Strava

Hướng dẫn lấy thông số Strava:
  • Client ID & Secret: Đăng nhập tài khoản Strava, truy cập Strava API Settings. Tạo một ứng dụng và điền trường Authorization Callback Domain là localhost (hoặc domain web của bạn). Sau khi lưu, bạn sẽ thấy Client ID và Client Secret.
  • Club ID: Truy cập trang Câu lạc bộ của bạn trên website Strava. ID chính là dãy số cuối cùng nằm ở trên thanh địa chỉ URL của trình duyệt (Ví dụ: URL là https://www.strava.com/clubs/1534169 thì Club ID của bạn là 1534169).
  • Cách lấy Cookie Strava (dành cho bộ cào Web): Đăng nhập tài khoản Strava trên trình duyệt máy tính. Nhấn phím F12 -> Chọn tab Application (Chrome/Edge) hoặc Storage (Firefox) -> Vào mục Cookies -> Bấm chọn https://www.strava.com -> Tìm cookie có tên _strava4_session -> Nhấp đúp chuột và copy toàn bộ giá trị của nó dán vào ô cấu hình phía trên.

 Sao Lưu Dữ Liệu Hệ Thống

            

                Toàn bộ dữ liệu của giải chạy (vận động viên, thành tích hoạt động, quy chế, phần thưởng) được lưu trữ trong tệp cơ sở dữ liệu SQLite cục bộ trên máy chủ VPS.
            

            
            
                 Tải Bản Sao Lưu (.db) Về Máy
            
            
            

                 Cơ chế tự động & Cách khôi phục:
                
• Sao lưu tự động: Hệ thống tự động sao lưu dữ liệu ngầm mỗi 24 giờ và lưu trữ xoay vòng 5 bản gần nhất trên ổ đĩa VPS để dọn dẹp dung lượng.
                
• Đề phòng VPS hỏng: Hãy chủ động tải bản sao lưu .db về máy tính cá nhân của bạn định kỳ để cất giữ ở một nơi an toàn.
                
• Cách khôi phục: Nếu VPS cũ bị hỏng hoàn toàn, bạn chỉ cần thiết lập VPS mới, sao chép file sao lưu .db đã tải về này vào thư mục gốc của dự án trên VPS mới, đổi tên file thành SSO_HC.db và khởi chạy ứng dụng. Toàn bộ dữ liệu sẽ được khôi phục nguyên vẹn 100%!
