Cài Đặt API Strava

Hướng dẫn lấy thông số Strava:
  • Client ID & Secret: Đăng nhập tài khoản Strava, truy cập Strava API Settings. Tạo một ứng dụng và điền trường Authorization Callback Domain là localhost (hoặc domain web của bạn). Sau khi lưu, bạn sẽ thấy Client ID và Client Secret.
  • Club ID: Truy cập trang Câu lạc bộ của bạn trên website Strava. ID chính là dãy số cuối cùng nằm ở trên thanh địa chỉ URL của trình duyệt (Ví dụ: URL là https://www.strava.com/clubs/1534169 thì Club ID của bạn là 1534169).
